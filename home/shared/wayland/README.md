# NixOS 桌面环境配置指南 (GNOME & Niri)

这套 NixOS 配置提供了一个双轨制的桌面体验：既有开箱即用、稳定全面的 **GNOME**，也有极简现代、键盘驱动的 **Niri (Wayland 卷轴式平铺窗口管理器)**。两者共享一套核心应用和底层系统配置。

---

## � 如何在 GNOME 和 Niri 之间切换？

由于 NixOS 的声明式特性，桌面环境是在系统启动（或登录）时选择的。您**不需要**修改代码来在两者间切换。

### 切换步骤：
1. **保存工作并注销**当前账号（GNOME 下点击右上角关机菜单 -> 注销；Niri 下按 `Super + X` 调用 wlogout -> 注销）。
2. 在 GDM 登录界面的用户选择页，点击**右下角的齿轮 ⚙️ 图标**。
3. 选择您想进入的桌面会话：
   - **GNOME**（或 GNOME on Wayland）
   - **Niri**
4. 输入密码正常登录即可。

> **💡 提示**：您在两个环境中使用的是同一个用户 (`echoyn`)，所有的文件、代码、浏览器记录都是完全互通的。

## 🌟 核心特性概览

无论是使用 GNOME 还是 Niri，这套配置都提供了一致的现代化工具链和 **Catppuccin Mocha** 暗色主题体验。

### 🛠️ 共享工具链
- **终端环境**: Ghostty (默认终端) + Fish shell + Starship (Catppuccin 提示符)
- **终端复用**: Zellij (替代 tmux)
- **版本控制**: Git + Delta (友好的 diff 查看器)
- **编辑器**: Neovim + Zed
- **输入法**: Fcitx5 + 雾凇拼音 (双系统无缝支持)

### 🎨 Niri 专属增强组件
当进入 Niri 会话时，会配套加载以下定制化组件：
- **状态栏**: Waybar（支持硬件监控抽屉、背光调节、音量滚轮）
- **应用启动**: Rofi（超大居中界面，纯键盘极速操作）
- **壁纸管理**: swww (守护进程) + 自定义 rofi 壁纸选择器
- **通知系统**: Mako
- **会话管理**: wlogout + swaylock

## ⌨️ Niri 常用快捷键 (Cheatsheet)

Niri 采用独特的"卷轴式"无尽向右平铺逻辑，学习这些快捷键可以极大提升效率。

### 基础启动
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Super + T` | 打开 Ghostty | 默认终端 |
| `Super + D` | 打开 Rofi | 搜索并启动所有应用 |
| `Super + Q` | 关闭当前窗口 | |
| `Super + Shift + /` | **显示完整快捷键** | Niri 官方按键列表浮层 |

### 辅助工具
| 快捷键 | 功能 |
|--------|------|
| `Super + Shift + W` | **壁纸选择器**（从 `~/Pictures/Wallpapers/` 选择并带预览） |
| `Super + X` | **电源/会话菜单** (注销/锁屏/重启/关机) |
| `Super + Alt + L` | 立即锁屏 |
| `Super + F1` | 重启 Fcitx5 输入法（遇到卡bug时非常有效）|
| `Super + F2` | 重启 Waybar 状态栏 |

### 窗口导航 (Vim 风格)
| 快捷键 | 功能 |
|--------|------|
| `Super + H / L` | 在当前列中，向左/向右切换窗口焦点 |
| `Super + J / K` | 在当前列中，向下/向上切换窗口焦点 |
| `Super + Ctrl + H/J/K/L` | 移动当前窗口的位置 |
| `Super + F` | 将当前窗口最大化占满整列 |
| `Super + Shift + F` | 真正的全屏模式 |

### 工作区与截图
| 快捷键 | 功能 |
|--------|------|
| `Super + 1~9` | 切换到指定工作区 |
| `Super + Tab` | 快速切回上一个工作区 |
| `Print` | 交互式框选截图 |
| `Ctrl + Print` | 截取全屏 |

## ⚙️ 系统维护与自定义

### 1. 更新系统配置

修改了 `~/nixos-config/` 下的任何 `.nix` 或 `.kdl` 文件后，通过以下配置好的 Fish alias 快速生效：

```bash
# 进入配置目录
nixcfg 

# 应用系统构建 (alias)
rebuild

# 如果修改了 flake.lock（比如你想升级系统内核或软件版本）(alias)
update
```

### 2. 管理壁纸

Niri 环境会自动从 `~/Pictures/Wallpapers/` 目录读取图片。
1. 将下载的图片 (`.jpg`, `.png`, `.webp`) 丢进该目录。
2. 每次 Niri 启动时会自动随机挑选一张。
3. 按 `Super + Shift + W` 可以随时预览并切换。

### 3. Niri 深度自定义

Niri 的所有配置都在 `home/shared/wayland/niri-config/` 目录下（按模块拆分）：
- `binds.kdl`：修改或添加快捷键
- `windowrule.kdl`：让某些应用默认开启浮动，或固定在某工作区
- `startup.kdl`：添加开机自启程序
- `theme.kdl`：调整边框缺口大小和光标主题

> **不用重启生效**：修改 `*.kdl` 文件保存后，Niri 会**热更新**并立即生效，如果语法写错了屏幕顶部会出现红色的错误提示，排错非常直观。
